## দুইটা নাম্বার এর যোগ

### in C
```c
#include<stdio.h>
int main(){

    int a,b;
    scanf("%d %d",&a,&b);
    printf("%d\n",a+b);
    return 0;
}
```

### in C++
```cpp
#include<iostream>
using namespace std;
int main(){

    int a,b;
    cin >> a >> b;
    cout << a+b << endl;
    return 0;
}
```
